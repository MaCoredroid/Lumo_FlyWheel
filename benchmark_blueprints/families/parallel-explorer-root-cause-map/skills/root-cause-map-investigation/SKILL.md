# Root Cause Map Investigation

Use this skill when the task is a read-heavy root-cause investigation that should be decomposed across explorer-style subagents.

## Required workflow

1. Split the problem into at least two non-overlapping explorer questions.
2. Build an evidence table, not just a narrative summary.
3. Treat renderer-facing symptoms as suspect, not authoritative.
4. Disprove at least one plausible false lead with repo evidence.
5. End with one bounded remediation plan and explicit non-goals.

## Deliverable checklist

- one adapter symbol
- one aggregation symbol
- one renderer symbol ruled out
- one failing assertion quote
- one contradictory artifact explained away
