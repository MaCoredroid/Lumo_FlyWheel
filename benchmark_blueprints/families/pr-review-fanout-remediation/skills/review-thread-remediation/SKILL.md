# Review Thread Remediation

Use this skill when the task involves unresolved PR review threads with mixed states.

## Required workflow

1. Classify every visible thread before editing code.
2. Treat stale summaries as hints, not authority.
3. Separate actionable work from duplicate closure and obsolete suggestions.
4. Tie reply text to thread ids and actual changes.

## Deliverable checklist

- `thread_triage.json` exists
- reopened regression is fixed
- regression test was added or updated
- docs match the final behavior
- replies never claim obsolete work was applied
