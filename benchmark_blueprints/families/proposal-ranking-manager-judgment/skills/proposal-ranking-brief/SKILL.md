# Proposal Ranking Brief

Use this skill when a task asks for a managerial ranking among several implementation proposals.

## Workflow
1. Compare proposals against the stated objective and real repo evidence.
2. Incorporate staffing, rollout, and regression constraints explicitly.
3. Rank all options, not only the winner.
4. Reject weaker proposals for concrete reasons.

## Anti-Patterns
- Choosing the fastest proposal without risk analysis.
- Ignoring staffing or rollout constraints.
- Writing a generic pros/cons list without a real decision.

## Done Signal
- The brief selects the best proposal and explains why the others lose.
