# Objective Delta Planning

Use this skill when a task asks for the best next intervention against a repo objective.

## Workflow
1. Compare allowed interventions against recent objective history.
2. Reject disallowed moves before optimizing for upside.
3. Weigh regression risk against expected delta.
4. Choose one next step and justify it clearly.

## Anti-Patterns
- Picking the highest upside move without governance checks.
- Returning a vague basket of options.
- Ignoring compounding effects from already-landed changes.

## Done Signal
- One concrete next intervention is selected with bounded delta and risk reasoning.
