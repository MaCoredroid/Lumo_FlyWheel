# Spec Citation Grounding

Use this skill when a benchmark task requires an evidence-grounded answer from a frozen RFC or spec corpus.

## Workflow
1. Search the local corpus before drafting conclusions.
2. Separate normative language from commentary or migration advice.
3. Cite every material claim with a pinned document location.
4. Call out the strongest contradictory or stale source explicitly.

## Anti-Patterns
- Answering from prior knowledge.
- Citing only one document when the task requires synthesis.
- Using a draft or migration memo as the primary authority.

## Done Signal
- The final memo is correct, pinned, and grounded in the frozen corpus.
