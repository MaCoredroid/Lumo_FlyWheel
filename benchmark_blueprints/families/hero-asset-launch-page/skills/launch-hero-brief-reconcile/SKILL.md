# Launch Hero Brief Reconcile

Use this skill when a launch-page hero must be aligned to a copy brief plus a pool of candidate images and responsive crops.

## Inputs
- Approved launch brief and alt-text guidance.
- Candidate hero assets and media metadata.
- Current desktop, tablet, and mobile renders.

## Workflow
1. Reconcile the brief against the current hero and candidate assets.
2. Confirm exact asset identity before tuning presentation.
3. Prefer shared media metadata and content sources over component-local literals.
4. Verify desktop, tablet, and mobile renders with fresh evidence.
5. Note any cache, optimization, or CDN-delivery risk.

## Guardrails
- Do not choose an asset by filename similarity alone.
- Do not fix tablet framing only with breakpoint CSS if shared focal metadata is wrong.
- Do not leave stale content or weak alt text in the source of truth.
