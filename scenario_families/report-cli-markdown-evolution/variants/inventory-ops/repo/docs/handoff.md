# Shift Handoff Layout

Shift leads read the handoff in this order:

1. Title
2. One-line queue summary
3. Top owner callout
4. Detailed sections table
5. Owner totals table

The owner totals block is the fastest way to see who is carrying the most
queued work into the next shift, so the order must stay deterministic.
When the watchlist is enabled, on-shift owners should remain visible in
that totals table even if they have zero queued items; a quiet queue needs
to be explicit rather than invisible.
